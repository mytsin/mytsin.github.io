Git常用命令速览

发表于 2019-11-16 更新于 2020-01-01 阅读次数：
本文字数： 364 阅读时长 ≈ 1 分钟
代码 注释
ssh-keygen -t rsa -C “your_email@example.com“ 设置SSH
git config –global user.name “your name” 设置用户名
git config –global user.email “your_email@youremail.com“ 设置邮箱
git init 初始化仓库
git status 查看是否存在改动
git add . 把文件添加到本地版本库
git commit 把文件提交到仓库
git remote add origin 你的远程库地址 关联到远程仓库
git pull –rebase origin master 远程库与本地同步合并
git push -u origin master 将最新的修改推送到远程仓库
git log 查看提交历史
git diff 比较工作区和暂存区
