如何使用VS code同时编辑多行，插入递增的数字?

有时在写代码，想要插入递增数字，如何实现这一功能？我们可以通过一个插件（increment-selection）来实现。

Step 1: 多光标操作

在VS code中有多种多行编辑的方法，根据自己的需要进行选择。

Win:

  1. Ctrl+Alt+上下箭头

  2. ALT+鼠标左键单击

  3. Shift+Alt+左键下拉

Mac: Option+Cmd+向下键

Step 2: 使用快捷键，实现递增

Win\Linux: Ctrl + Alt + I

Mac: Cmd + Alt + I

值得注意的是，默认插入的递增数字中的起始数字是0。如果想改变起始数字，在多行编辑前将第一项内容更改为想要的数字，多行选中后再使用快捷键。

anaconda用法：

查看已经安装的包：
pip list 或 conda list

安装和更新：
pip install requests或pip install requests --upgrade

conda install requests或conda update requests
